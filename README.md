## 資料改寫

### i18n 中英文檔
- 由 src / lang 的中英文資訊產生
  - 目前只支援 en.json(英文) / zh.json(中文)

### store 資料
- 畫面資料由 src / stores / data.ts 統一畫面資料處理
  - 首頁／自傳／技能樹／專案作品／履歷 內容 

## 建置
```
npm run build
```
- dist 資料夾就是完成的專案
